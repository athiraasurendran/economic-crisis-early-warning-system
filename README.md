# Economic Crisis Early Warning System for India 🇮🇳

## Objective

The goal of this project is to build a machine learning-based early warning system that predicts the risk of an economic crisis in India.

This system analyzes key macroeconomic indicators such as GDP growth, inflation rate, unemployment rate, interest rates, and stock market trends to generate a "Crisis Risk Score" and classify the economic condition as:

- Safe
- Warning
- Crisis

This project combines economics knowledge with data science techniques to provide real-world insights for policymakers, investors, and businesses.

## Problem Statement

Economic crises can have severe impacts on employment, businesses, and financial stability. Early detection of economic downturns is crucial for taking preventive measures.

However, traditional analysis methods are often reactive rather than predictive.

This project aims to develop a predictive model that can identify early signals of an economic crisis using historical macroeconomic data.

## Features

- GDP Growth Rate
- Inflation Rate (CPI)
- Unemployment Rate
- Interest Rates (Repo Rate)
- Stock Market Index (NIFTY/SENSEX)
