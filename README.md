# Economic Crisis Early Warning System for India 🇮🇳

## Objective

The goal of this project is to build a machine learning-based early warning system that predicts the risk of an economic crisis in India.

This system analyzes key macroeconomic indicators such as GDP growth, inflation rate, unemployment rate, interest rates, and stock market trends to generate a "Crisis Risk Score" and classify the economic condition as:

- Safe
- Warning
- Crisis

This project combines economics knowledge with data science techniques to provide real-world insights for policymakers, investors, and businesses.
